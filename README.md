# React Native WebView + Video Player App

A mobile app that shows websites, plays videos, and sends notifications.

## What Does This App Do?

### 1. WebView Page

Shows a website inside the app (like a mini browser)

- 2 buttons to send notifications
- Notifications appear after 2-3 seconds

### 2. Video Player Page

Plays streaming videos with controls

- Play/Pause button
- Skip forward 10 seconds
- Mute/Unmute sound
- Fullscreen mode

### 3. Notifications

- Bell icon at top shows notification count
- Tap bell to see all notifications
- Tap notification to go to video page

### 4. How to run this project

- Git clone Github url
- npm install
- npx expo start
- press a for android and i for ios

## What Technologies Are Used?

- **Expo** - Makes React Native easier
- **React Native** - Build mobile apps with JavaScript
- **React Navigation** - Switch between pages
- **WebView** - Show websites in the app
- **Expo AV** - Play videos
- **Expo Notifications** - Send notifications
- **Lucide Icons** - Pretty icons
